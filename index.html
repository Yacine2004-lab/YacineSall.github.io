Bonjour, je suis Yacine

À propos de moi

Je suis Yacine, étudiante en 2ème année de Licence en Génie Informatique à UNIPRO. Je m'intéresse à la programmation web, au développement logiciel et à la conception d’interfaces intuitives. Curieuse et déterminée, j'aime apprendre et relever des défis techniques. Ce portfolio regroupe quelques projets qui illustrent mes compétences et ma passion pour l’informatique.

Mes Compétences

  • HTML / CSS / JavaScript
  • Python (Tkinter, PyQt, POO)
  • Base de données SQLite
  • Git & GitHub
  • Modélisation UML

Mes Projets

Application de gestion de contacts

Application Python avec interface graphique Tkinter, permettant d’ajouter, modifier et supprimer des contacts, enregistrés localement dans une base SQLite. L’app inclut une protection par clé de produit et un installeur.

Site de fast-food avec UML

Projet universitaire modélisé avec UML. Il gère la prise de commande, le stock de produits et la livraison. Utilisation de cas d'utilisation, diagrammes de classes et séquences pour modéliser l’architecture du système.

Portfolio personnel HTML/CSS

Un site web moderne pour présenter mes compétences, projets et me contacter facilement. Déployé avec GitHub Pages avec un design responsive et des animations douces.

Mini-blog en Python Flask

Un blog minimaliste codé avec Flask, permettant de publier, éditer et supprimer des articles. Utilise une base SQLite, un moteur de template Jinja2 et du CSS responsive.

Application de suivi de dépenses

Application permettant de suivre les dépenses personnelles, d'ajouter des catégories et d'analyser les dépenses mensuelles avec des graphiques en Python et matplotlib.

Jeu de mémoire en JavaScript

Un jeu de mémoire interactif où l'utilisateur doit retrouver des paires de cartes. Développé en JavaScript avec du CSS pour le design et la gestion du score.
